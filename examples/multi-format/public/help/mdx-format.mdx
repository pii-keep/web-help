# MDX Format Example

This is an example of a help page written in **MDX**. You can use React components inline with Markdown for rich, interactive help experiences.

<HelpCallout type='info'>
  MDX lets you embed React components directly in your help content!
</HelpCallout>

## Usage

To use MDX in your help system, configure your content loader to parse `.mdx` files.

<HelpCodeBlock language="tsx">{`
import { HelpContent } from '@piikeep/web-help'

export const MyHelpPage = () => (
    <HelpContent>
        {/* MDX content will be rendered here */}
    </HelpContent>
)
`}</HelpCodeBlock>

## Features

- Markdown + JSX
- Interactive components
- Frontmatter support

# Markdown Format

This article is written in **Markdown** (.md) format.

## What is Markdown?

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents.

## Features

- **Simple**: Easy to read and write
- **Portable**: Works everywhere
- **Flexible**: Supports HTML when needed

## Code Example

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Lists

Ordered list:

1. First item
2. Second item
3. Third item

Unordered list:

- Bullet point
- Another point
- Final point

## Links and Images

You can include [links](https://example.com) and images in Markdown.

> **Tip**: Markdown is the most popular format for help documentation!

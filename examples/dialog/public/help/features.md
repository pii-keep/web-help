# Features Guide

Learn about the key features of this help system.

## Navigation

The help system provides automatic navigation:

- **Sidebar Navigation**: Browse all help articles by category
- **Breadcrumbs**: See where you are in the documentation
- **Previous/Next**: Navigate between articles sequentially

## Content Formats

You can write help content in multiple formats:

- Markdown (`.md`)
- MDX (`.mdx`) - Markdown with JSX
- JSON (`.json`)
- CSV (`.csv`)

## Search

Use the search feature to quickly find what you're looking for:

1. Click the search box
2. Type your query
3. Select from the results

## Customization

The library is completely headless, which means:

- You control all the styling
- Components provide semantic class names
- Data attributes for state management
- Full flexibility in layout and design

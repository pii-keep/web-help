# Component Styling

Learn how to style help components using the baseline CSS.

## Navigation

The sidebar navigation is fully styled with:

- Collapsible categories
- Active state highlighting
- Hover effects
- Smooth transitions

## Content Area

The main content area includes styling for:

- Typography (headings, paragraphs, lists)
- Code blocks with syntax highlighting
- Tables
- Images and media
- Callouts and alerts

## Breadcrumbs

Breadcrumb navigation shows your current location:

- Clickable parent links
- Separator icons
- Current page indicator

## Pagination

Previous/Next navigation at the bottom of each article:

- Clear labels
- Article titles
- Arrow indicators
- Hover states

## Responsive Design

The baseline CSS is fully responsive:

- Mobile-friendly navigation
- Collapsible sidebar on small screens
- Touch-friendly controls
- Readable typography across devices
